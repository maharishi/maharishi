On-line Zip

This application is used to create zip of files which are available on other servers. This will help the users to download the files in a zip format so that the bandwidth used is less. This application uses an open source zip library SharpZipLib to acheive compression. In future I might be adding other formats of compression which this library supports. Zipping of file using upload method support a maximum of only 4 Mb.
